# Spotify-2023

Exploratory Data Analysis (EDA) sur le dataset Spotify 2023.

## Contenu
- Analyse des tendances musicales 2023
- Visualisations des caractéristiques audio
- Statistiques descriptives

## Dataset
Source : Spotify 

## Librairies utilisées
- Pandas
- Matplotlib / Seaborn
- NumPy

## Objectif
Explorer les tendances musicales de 2023 à travers les données Spotify.
